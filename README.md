# Ground Control LLC website (Next.js)

## Run locally (Windows)
1. Install Node.js LTS (18+)
2. In this folder:
   - `npm install`
   - `npm run dev`
3. Open http://localhost:3000

## Deploy on Vercel
- Import the GitHub repo in Vercel
- Framework: Next.js (auto-detected)
- Build Command: `npm run build`
- Output: (leave default)
